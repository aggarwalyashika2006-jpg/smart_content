# Smart Content Generator for Social Media Campaigns 🚀

## 📌 Project Overview
This project demonstrates a no-code, AI-powered workflow to create and manage social media campaigns.  
It uses tools like ChatGPT, Canva, and Google Sheets to generate content, design visuals, schedule posts, and analyze performance.

---

## 🎯 Objective
To design a simple and efficient system that helps students and beginners create engaging social media content without coding.

---

## 🛠️ Tools Used
- ChatGPT – Content ideas, captions, hashtags  
- Canva – Visual design (carousel & reels)  
- Google Sheets – Content planning and scheduling  
- Buffer / Meta Business Suite – Scheduling (conceptual)  

---

## 🔄 Workflow

1. Define Campaign Goal & Audience  
2. Generate Content Ideas  
3. Create Captions & Hashtags using AI  
4. Design Visual Content using Canva  
5. Schedule Posts using Google Sheets  
6. Analyze Engagement Metrics  

---

## 📊 Features
- AI-generated content  
- No coding required  
- Structured content calendar  
- Visual content creation  
- Engagement analysis
